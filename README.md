# Great-Lakes-WW3

## Description
Plot maps of Great Lakes Wave-Watch III data from GLERL to preset areas
## Dependencies 
Dependencies include: Xarray, Numpy, Metpy, Matplotlib, Cartopy, Datetime, Requests, PyProj, Pandas, Sys, Warnings
